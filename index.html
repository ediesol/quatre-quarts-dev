    • quatre quarts is a timer emphasizing which share of a given timespan is left, rather than only focusing on time duration.

    Some less common mobile browsers stop the timer from ringing. If you encounter this, try adding an exception enabling audio and/or video autoplay for this site.

    tl;dr:

    quatre quarts uses Cabin Analytics, a privacy focused analytics service that does not collect any personal data. Its only purpose is to provide general information on how and how much quatre quarts is used.

    how it works:

    Cabin collects anymous general usage data, for example: how many people visit the page, for how long, which device and browser they use, which country they connect from. These data points are stored separately from each other, meaning there is no joint data about a specific user, only general statistics.

    Cabin does not collect any IP address, nor create unique user identifyers, nor use cookies.

    How I use the collected data: my aim is to have some basic information on usage while collecting as little data as possible. So far this is primarily out of curiosity, and possibly to improve quatre quarts as well.

    Collected data is stored for 30 days.

    For more information or an example of the data collected you can visit Cabin's website or documentation.
